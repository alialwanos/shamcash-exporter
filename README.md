# ShamCashExporter (بدون روت - Android 14)

هذا مشروع Android Studio (Kotlin) يستخدم **Accessibility Service** لفتح تطبيق الهدف،
محاولة الانتقال إلى تبويب **التحويلات/العمليات** وقراءة العناصر الظاهرة على الشاشة ثم تصديرها كـ **JSON**.

## مهم
- بدون روت ✅
- يحتاج تفعيل خدمة إمكانية الوصول يدويًا من الإعدادات (هذا شرط أندرويد).
- يقرأ فقط البيانات الظاهرة على الشاشة (لا يقرأ API أو بيانات مخفية).

## الاستخدام
1) افتح المشروع في Android Studio.
2) شغل التطبيق على هاتفك.
3) من داخل التطبيق اضغط: **افتح إعدادات إمكانية الوصول** ثم فعّل:
   ShamCash Exporter -> Allow / تمكين.
4) تأكد من اسم الباكيج (Package) للتطبيق الهدف في الخانة (افتراضي: com.shmacash.shamcash).
5) اضغط **ابدأ**:
   - سيحاول فتح التطبيق الهدف
   - ثم سيحاول الضغط على "التحويلات"
   - ثم سيجمع العناصر المرئية ويصدر JSON (one-shot)
6) سترى JSON داخل التطبيق ويمكنك نسخه.

## تعديل سريع
إذا كانت أسماء التبويب مختلفة، عدل القائمة داخل:
`ShamAutomationService.kt` -> `navTargets`

## ملاحظات تقنية
- التحليل يعتمد على Heuristics (تجميع نصوص ورصد أنماط مثل #ID و التاريخ والمبلغ).
- قد تحتاج تخصيص parsing حسب الـ UI الحقيقي.
